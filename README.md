# Framework Padawans

## Carlos Magalhães Silva

Na implementação das telas a linguagem foi utilizado ReactJS. Para instalar os módulos necessários para a execução do programa:

```
$ yarn install
```

Para executar o programa:

```
$ yarn start
```
